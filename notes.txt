Procedure to Create New App
1.clone the repositary from github
  git clone https://github.com/santoshmswami/batchrecordingapp.git
2.create angular project
  ng new batchrecordingapp
3.install bootstrap
  npm i bootstrap  or (npm install bootstrap --save)
  "styles": [
     "node_modules/bootstrap/dist/css/bootstrap.min.css",
     "src/styles.css"   
    ],
    "scripts": [
    "node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"
    ]
4.open the app.route.ts define the defauly routes
5.we are going to use template from so we need to bind
